# Backend Task Manager API

## 🚀 Overview
A scalable backend system with authentication, role-based access, and CRUD operations for tasks. Includes a basic frontend to interact with APIs.

---

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

---

## ✨ Features
- User registration & login
- JWT-based authentication
- Role-based access (User/Admin)
- CRUD operations for tasks
- Protected routes
- Basic frontend integration

---

 ├── middleware/
 ├── config/
